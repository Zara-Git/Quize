Quiz App – Inlämningsuppgift 1

Beskrivning
En enkel quizapplikation byggd med HTML, CSS och JavaScript. Användaren väljer ett quiz på startsidan, svarar på frågor och får resultat direkt.

Krav som uppfylls

Startsida med flera quiz

Varje fråga har minst 4 svarsalternativ (radio/kryssfrågor)

Självrättande test med resultat efter alla frågor

Alla quiz lagras i localStorage i JSON-format

Det går att skapa och spara nya quiz som visas på startsidan

Logik (kort)

Quiz-data hämtas från localStorage och ritas ut dynamiskt med JavaScript.

Användarens val kontrolleras mot isCorrect och poängen räknas ihop.

Resultatet visas i en modal/popup.

Körning

Öppna index.html i webbläsaren

Välj ett quiz och klicka “Starta”

Svara på alla frågor och klicka “Rätta quiz”

Tekniker
HTML, CSS, JavaScript, localStorage (JSON), Bootstrap Icons
